# Code of Conduct

## Our Commitment

This project is maintained by an individual developer and shared publicly as a product demo. Everyone who interacts with this repository — through issues, discussions, or pull requests — is expected to engage respectfully and constructively.

---

## Expected Behaviour

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Accept constructive feedback gracefully
- Focus on what is best for the project
- Show courtesy and professionalism in all interactions

---

## Unacceptable Behaviour

- Harassment, abuse, or discriminatory language of any kind
- Personal attacks or insults
- Sharing others' private information without permission
- Any conduct that would be considered inappropriate in a professional setting

---

## Scope

This Code of Conduct applies to all project spaces — GitHub Issues, Pull Requests, Discussions, and any other channel where this project is represented.

---

## Enforcement

Instances of unacceptable behaviour may be reported by contacting the project maintainer directly via [LinkedIn](https://www.linkedin.com/in/mohith-sai-gorla-75930318b/). All reports will be reviewed promptly and handled with discretion.

The maintainer reserves the right to remove comments, close issues, or block contributors who violate this Code of Conduct.
